# Anik Infra Trading Business Website

## Overview

Anik Infra Trading is a mining and logistics company based in Kitwe, Zambia. This project involved the development of a static business website for the company, focusing on showcasing the company's services, background, and expertise in the mining and logistics industry. The website was designed to be primarily front-end with minimal backend functionality.

### Features

- **Company Overview**: Detailed information about Anik Infra Trading, including company history, mission, and vision.
- **Services Showcase**: Highlighting the company's services in mining and logistics with descriptive content and images.
- **Contact Information**: Contact details and location map for easy communication and location tracking.

## Project Status

**Note:** A future update will include the integration of an SMTP mail function to allow customers to send email messages to the company directly from the website.

- **Front-end**:

## Features Detail

### Company Overview

The website provides comprehensive information about Anik Infra Trading, detailing the company's history, mission, and vision. This section aims to give visitors a better understanding of the company's background and values.

### Services Showcase

Anik Infra Trading's services in mining and logistics are prominently featured on the website, with descriptive content and images illustrating the company's expertise and capabilities in these sectors.

### Contact Information

The website includes contact details for Anik Infra Trading, allowing visitors to easily reach out to the company for inquiries, collaborations, or any other purposes. A location map is also provided for visitors to find the company's office location easily.

## Getting Started

### Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/anik-infra-trading-website.git
```

### Development

1. Start a local server (e.g., using PHP's built-in server):

```bash
php -S localhost:8000
```

2. Open [http://localhost:8000](http://localhost:8000) in your web browser to view the website.
